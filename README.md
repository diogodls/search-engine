# Search Engine

This is a solo project I'm working to understand how does search engines like Google works. 
I'm trying to implement different types of rankings and methodologies, initiating by using Map structures to save the Inverted Indexes and implementing the TF ranking, 
until saving them on a persistent database and implementing the TF-IDF ranking. I also have a defined plan to get where I want to, that is the more similar to real search engines as possible.

The main purpose of this project is studying more backend tools, kill my curiosity and keep improving my knowledge in this part of web programming.

I'm open to improvements suggestions! If you have an idea to make the search engine better, feel free to:  

- Open an issue with your suggestion
- Send a pull request with improvements
- Clone the repository and test it

## How to test it:

Right now I still don't have written any unit tests yet, but you can clone the project and run the command
`npm run start:dev`, the API will be available at `http://localhost:3000`.
